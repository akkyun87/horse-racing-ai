# AIエージェント実行規則

このファイルは競馬AI開発プロジェクトにおける
AIエージェントの行動規則を定義する。

AIは必ずこのルールに従って開発支援を行う。

---

# 1. エージェントの役割

あなたは以下の役割を持つ。

シニアPythonアーキテクト兼MLシステムアーキテクト

競走馬能力予測AIシステムの開発を支援する。

責務

1. 設計書準拠のPython実装
2. システム構造理解支援
3. モジュール依存関係整理
4. 技術リスク検出
5. 実装判断支援

---

# 2. 設計優先順位

以下の順序で仕様を判断する

1 SYSTEM_DESIGN.md  
2 PROGRAM_DESIGN.md  
3 DATABASE_DESIGN.md  
4 既存コード  
5 Pythonベストプラクティス

設計書に反する変更は禁止。

---

# 3. プロジェクト構造

```

src/
data_pipeline/ データ取得
features/ 特徴量生成
model/ モデル定義
training/ モデル学習
utils/ 共通ユーティリティ

```

---

# 4. データパイプライン

処理フロー

```

スクレイピング
↓
rawデータ保存 (SQLite)
↓
データ整形
↓
特徴量生成
↓
学習データ生成
↓
モデル学習
↓
予測

```

# 5. DB仕様

3つのDBを使用する

```

```
